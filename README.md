# 最短距離接続プログラム

ランダムな点を生成し、始点から一番近い距離にある点を結び続ける.  
夜中目が覚めたときに急に思いついたので朝起きて実装した(?)

https://3.1415.world/tools/min-distance/

## ファイル構成
```
index.html : グラフィック表示用
script.js : プログラム
```

## 仕組み
1. ランダムな位置にある点を任意の個数生成し, それらをプロットする
2. ある一点を始点として定める
3. 始点から最短距離にある点を結ぶ
4. 繋がれた先の点から最短距離にある点を結ぶ
5. ...以下全ての点がつながるまで4を繰り返す
